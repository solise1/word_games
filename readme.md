# Word games using Django

This repo has one game already and I'll expand it a bit more before moving on to a new project.

For now you can use it to guess words (akin to Wordle).

## Instructions to run

- Run migrations (required for sessions):

```python
python manage.py migrate
```

- Run server normally:

```python
python manage.py runserver
```
